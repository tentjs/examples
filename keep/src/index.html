Keep is a built-in attribute that, when set to true, will cache the children of the element after first render.

This is useful when using external libraries that generate HTML that Tent is not in control of. This could be something like Lucide.

keep was introduced in @tentjs/tent@0.0.33-2 as an experimental feature. It is not stable, and may change in the future.
