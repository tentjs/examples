⏳ Counter

This is a simple counter example. It demonstrates the fundamentals of ⛺ Tent.

It uses state, dynamic content1 and sets up a button with a regular onclick event.

[1]dynamic content is a string with 1 or more placeholders that will be evaluated. In this case, the placeholder is {state.count} and the string is Count: ${state.count}. It is nothing more than a regular JavaScript template literal.
